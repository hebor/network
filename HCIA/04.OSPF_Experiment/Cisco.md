# OSPF基本命令

<table>
    <tr>
    	<td>router ospf 1</td>
        <td>启动OSPF进程</td>
    </tr>
    <tr>
    	<td>router-id 1.1.1.1</td>
        <td>配置OSPF路由器的Router ID</td>
    </tr>
    <tr>
        <td>show ip ospf neighbor</td>
        <td>查看OSPF邻居表</td>
    </tr>
    <tr>
        <td>show ip ospf database</td>
        <td>查看OSPF数据库</td>
    </tr>
    <tr>
        <td>show ip route ospf</td>
        <td>查看OSPF路由表</td>
    </tr>
</table>
